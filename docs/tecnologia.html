Tecnologia — Loja

Esteira de Organização de Trabalho: Python (Baseline) na oficina; HTML/CSS/JS na sala (pública). Auditável e desenhado para privacidade (LGPD). Não vendemos.

Oficina: Python (Baseline)

Os 6 assistentes (iniciar trabalho, validar, sincronizar/transportar, validar entrega face/gram/retro) são em Python. Versão Baseline (Microsoft Store). Pacote caracore_mkt e pasta scripts/ neste repositório. Planilha CSV; mensagens para WhatsApp, Telegram e e-mail. O fluxo é auditável: cada etapa pode ser verificada.

Sala Cara Core: pública

A Sala Cara Core é pública. O conteúdo (planos, campanhas, prompts IA, feedback) está em caracore-site/sala — HTML, CSS, JS estáticos. Estrutura assets/css, assets/img, assets/js. Qualquer pessoa pode acessar. Uso fora das normas éticas e de privacidade pode resultar em bloqueio de acesso.

Sala: HTML, CSS, JS

Conteúdo da Sala (planos, campanhas, prompts IA, feedback) é estático: HTML, CSS, JS. Páginas em sala/ (caracore-site/sala). Acesso público.

Privacidade e LGPD

O sistema foi desenhado para respeitar a privacidade (LGPD). Dados são tratados com finalidade clara; não há coleta desnecessária. O desenho auditável e a transparência são pilares: quem opera e a supervisão podem verificar os passos. Uso antiético bloqueia o acesso à sala.

Não vendemos

O Seed e o Mkt são da Cara Core; não vendemos. Stack interna para campanhas orgânicas e oficina.
