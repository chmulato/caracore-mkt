O que é o Cara Core Mkt?

A Sala Cara Core é um ambiente de aprendizado e gestão operacional. Scripts de automação simples para um fluxo linear, organizado e justo. Centraliza a Sala Cara Core (planos e campanhas — pública), o Portal (acesso controlado) e a oficina (scripts em Python — este repositório) que sincronizam, organizam e validam. O foco é a execução correta de processos; o sistema é auditável e desenhado para privacidade (LGPD). Uso fora das normas éticas bloqueia o acesso. Imagens e artefatos por e-mail a suporte@caracore.com.br. Não é produto comercializado.

Onde está cada coisa

Sala (pública): repositório do site, pasta sala/. A Sala Cara Core é pública. Oficina (scripts): este repo — caracore-mkt. Delivery (matriz): caracore-site/delivery/mkt. Loja (esta página): caracore-mkt/docs — mesmo layout do delivery.

Requisitos e links

Python Baseline (Microsoft Store) para rodar os scripts. Na raiz do repo: README.txt, CTO_VALIDACAO_MKT.txt, LICENSE (MIT).
