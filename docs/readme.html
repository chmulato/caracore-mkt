O que é o CaraCore MKT?

Ferramenta de uso interno da Cara Core: Sala de Notícias (planos e campanhas — pública), oficina (6 assistentes digitais em Python — este repositório) e canal de feedback. O diferencial competitivo está no valor da execução; os robôs protegem o tempo de quem opera. Imagens e artefatos por e-mail a suporte@caracore.com.br. Não é produto comercializado.

Onde está cada coisa

Sala (pública): repositório do site, pasta sala/. A Sala de Notícias é pública. Oficina (scripts): este repo — caracore-mkt. Delivery (matriz): caracore-site/delivery/mkt. Loja (esta página): caracore-mkt/docs — mesmo layout do delivery.

Requisitos e links

Python Baseline (Microsoft Store) para rodar os scripts. Na raiz do repo: README.txt, CTO_VALIDACAO_MKT.txt, LICENSE (MIT).

Não vendemos

O Seed e o MKT são da Cara Core; não vendemos. Documentação para referência e uso interno.
