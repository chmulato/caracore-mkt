# Cara Core MKT — Documentação (loja)

Repositório da **oficina** da Sala de Notícias: scripts em Python para iniciar trabalho, validar, transportar e validar entrega (face, gram, retro). O Seed e o MKT são da Cara Core; não vendemos. Esta é a **loja** — online, livre para passear e conhecer.

## Estrutura da pasta `docs/`

```
docs/
  README.md                 — esta página (entrada da documentação no GitHub)
  assets/
    img/
      sala_de_noticias.png  — imagem que representa a Sala de Notícias (convite "entre em casa")
      prompt_img.txt        — prompt para gerar a imagem da loja (IA)
```

## O que tem neste repositório

| Pasta / arquivo | Descrição |
|-----------------|-----------|
| **scripts/** | 6 scripts Python: iniciar_trabalho, validar_trabalho_executado, validar_sala, transportar_para_salas, validar_entrega_face, validar_entrega_gram, validar_entrega_retro |
| **caracore_mkt/** | Pacote Python: config, planilha, mensagens, validar, util |
| **CTO_VALIDACAO_MKT.txt** | Validação CTO: automação, saída acionável, custo, escalabilidade |
| **README.txt** | Instruções para rodar (Python Baseline, caminhos, variáveis de ambiente) |
| **docs/** | Documentação e assets para GitHub (esta pasta) |

## Imagens

- **sala_de_noticias.png** — Usada no convite "Entre em casa" no delivery (Cara Core MKT). Representa a Sala de Notícias.
- **prompt_img.txt** — Prompt para gerar a imagem que representa a loja (oficina aberta, livre para conhecer). Use em IAs de imagem (DALL·E, Midjourney, etc.) se quiser criar ou atualizar a arte.
